Au ccours de cette formation nous allons avoir 2 matières :

L'intégration

Le HTML :

Le HTML sert à créer le cconenu du site : elle va contenir uniquement la partie textuelle du site.

Le CSS :

Le CSS va nous permettre d'ajouter la couche graphique à notre site : les couleurs, les formes, le fond, etc.

Le dévloppement

Le Javasccript :

Javasccript est un langage de programmation de script employé pour rendre une page web interactive, le comportement de la page et son apparence pourra varier en fonction de certains paramètres.

Le PHP :

Le PHP est un langage de programmation libre, utilisé pour générer des pages web dynamiques, qui sont donc modifiés en fonction de certains informations (heure, nom de l'utilisateur, formulaire rempli par l'utilisateur etc).

MySQL :

MySQL est un système de bases de données, il permet d'enregister des informations rempli par l'utilisateur (exemple :formulaire de création de compte) pour ensuite être ré-utilisé au moment approprié (ex: connexion à un site internet avec identifiant et mot de passe).
